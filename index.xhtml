Introduction

This website intends to introduce the key concepts behind web development. Namely, the 'three pillars' that go into construction of any website: content, presentation, and behaviour; and the technologies through which these are achieved: HTML, CSS, and JavaScript, respectively. The content of this site aims to give a brief overview of what each of these technologies is, their individual contributions to a completed website, and an overview of how the mark-up / code itself is written and the functionality it adds.

Content - HTML

The content of a website is achieved through HTML (hypertext markup language) files. These are where your site's content is 'marked up' using 'tags' and 'attributes' which adhere to the HTML standard.

Presentation - CSS

The presentation of a website is achieved through CSS (cascading style sheets). These allow you to assign attributes like font, backgrounds, colours, and positioning to elements of your HTML content file.

Behaviour - JavaScript

The behaviour of a website is achieved via JavaScript. It is a powerful scripting language which allows you to define functions with which you can manipulate your HTML and CSS code.
